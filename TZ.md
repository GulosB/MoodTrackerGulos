\# Техническое задание: MoodTrackerGulos



\## 1. Цель проекта

Веб-приложение для анализа настроения по дневниковым записям с визуализацией динамики.



\## 2. Роли пользователей

\- Гость (просмотр формы и графика)

\- Администратор (управление записями в админке)



\## 3. Модели данных

\- `MoodTag`: id, name (Позитив/Негатив/Нейтрально)

\- `DiaryEntry`: id, text, mood\_tag (FK), created\_at

\- `MoodStat`: date, positive\_count, negative\_count, neutral\_count



\## 4. Ключевой функционал

\- Форма для ввода текста

\- Автоматическое определение настроения по ключевым словам

\- Построение сглаженного графика динамики настроения с заливкой



\## 5. Аналитика

Использование Pandas, Matplotlib, Scipy для сглаживания и визуализации.
