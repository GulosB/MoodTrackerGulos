\# MoodTrackerGulos — Дневник настроения с аналитикой



Сервис определяет настроение по тексту (позитив, негатив, нейтрально) и строит сглаженный график динамики настроения.



\## Технологии

\- Python 3.14

\- Django 5.x

\- Pandas, Matplotlib, Scipy

\- SQLite

\- Bootstrap 5



\## Как запустить локально

1\. Клонируйте репозиторий

2\. Создайте виртуальное окружение: `python -m venv venv`

3\. Активируйте: `venv\\Scripts\\activate` (Windows)

4\. Установите зависимости: `pip install -r requirements.txt`

5\. Выполните миграции: `python manage.py migrate`

6\. Запустите сервер: `python manage.py runserver`
